Системата КСК 2017 е на Symfony 3.1.
Във view-частта на проекта (twig-овете) е използван Twitter Bootstrap и CSS.
Изработка: Мария Пенелова

Реализирани са следните функционалности:

1.) Регистрация на кандидат-студент в системата
    - с BotDetect Symfony 3 CAPTCHA
    - с валидация на входните данни на Symfony 3.1
    - с потвърждаване на регистрацията чрез изпращане на линк за активация на e-mail адреса на кандидат-студента. 
    След потвърждаване на регистрацията се активира акаунтът на кандидат-студента.
    
2.) Вход в системата
3.) Смяна на парола на кандидат-студент
4.) Профил на кандидат-студент
   - с валидация на входните данни на Symfony 3.1
